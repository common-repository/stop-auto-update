This plugin simply stop WordPress from automatically updating for you.

== Description ==

This plugin is not for most WordPress users. It will stop WordPress from automatically updating witch is not recommended. The only reason you might need this plugin is if you have modified your WordPress core files and you want WordPress to stop overwriting them with their automatic security updates. Otherwise you should probably just let WordPress auto update.

Updated April-19th

== Installation ==

1. Upload the plugin files to the Wordpress `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
